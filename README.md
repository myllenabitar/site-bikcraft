## 📖 Descrição do Projeto

Este projeto foi criado como parte dos exercícios do curso da **Origamid**, com incrementos e personalizações baseados nas minhas preferências pessoais. A experiência prática incluiu:

- **Estruturação Semântica**: Utilizando boas práticas com **HTML5**.  
- **Estilização Responsiva**: Aplicação de **CSS3** para garantir adaptabilidade em diferentes dispositivos.  
- **Interatividade Básica**: Implementações simples de **JavaScript** para funcionalidades adicionais.
